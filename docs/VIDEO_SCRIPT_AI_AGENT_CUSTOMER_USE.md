# CapeControl AI Agent Integration — Customer Instructional Video Script

**Duration:** 3:45 (approx.)
**Format:** Screen-capture walkthrough with voiceover
**Tone:** Professional, confident, focused on simplicity and efficiency

---

## SCENE 1 — Opening (0:00–0:20)

**VISUAL:** CapeControl logo animation → fade to the login screen at `cape-control.com`, then a smooth transition into the Dashboard.

**VOICEOVER:**

> Welcome to CapeControl — your intelligent operations platform.
>
> In the next few minutes, we'll show you how an approved AI agent works inside CapeControl: how to find one, authorise it, put it to work, and monitor it — all from the same dashboard you already use every day.

---

## SCENE 2 — The Dashboard at a Glance (0:20–0:50)

**VISUAL:** Full Dashboard view. Cursor highlights the **Welcome Header** with system status badge showing "Operational." Pan to the **Usage Progress Card** showing API calls, agent count, and storage meters. Brief hover over the **Activity Feed** showing recent events.

**VOICEOVER:**

> Here's your Dashboard. At the top you can see your system status is operational, and over here, your Usage Progress card tracks everything at a glance — API calls, active agents, and storage.
>
> The Activity Feed on the right gives you a real-time timeline of every action on the platform — logins, agent runs, evidence exports — all logged automatically.
>
> Now, let's bring an AI agent into the picture.

---

## SCENE 3 — Browsing the Agent Marketplace (0:50–1:30)

**VISUAL:** Click **Marketplace** in the sidebar. The Marketplace page loads showing the stats banner (published agents, total deployments, active users), category filter grid, and agent card listing.

**VOICEOVER:**

> Click **Marketplace** in your sidebar. This is your Agent Marketplace — a curated catalogue of approved, published AI agents ready to deploy.
>
> At the top, you can see how many agents are available and how many deployments are running across the platform.

**VISUAL:** Click the **Analytics** category to filter. A few agent cards appear. Mouse hovers over one card showing its name, description, version, and category badge.

**VOICEOVER:**

> You can filter by category — Automation, Analytics, Security, Monitoring, and more — or use the search bar to find exactly what you need.
>
> Each agent card shows its name, a brief description, current version, and when it was last published. Let's take a closer look.

---

## SCENE 4 — Reviewing Agent Details and Permissions (1:30–2:10)

**VISUAL:** Click **View Details** on the "Data Analyst" agent card. The detail modal opens showing agent author, description, permissions list, capabilities/tags, rating summary (star display), and reviews.

**VOICEOVER:**

> When you open an agent's details, you'll see exactly what it does, who built it, and — critically — what **permissions** it requires.

**VISUAL:** Cursor highlights the permissions list. A brief callout graphic appears: *"The AI agent only has the permissions you grant it."*

**VOICEOVER:**

> CapeControl enforces a strict permission model. An AI agent can only access the data and actions you've explicitly authorised. There are no hidden capabilities — what you see here is exactly what the agent can do.

**VISUAL:** Scroll down to the rating panel showing star distribution and a couple of community reviews.

**VOICEOVER:**

> You can also check ratings and reviews from other users on your platform before you decide to install.

---

## SCENE 5 — Launching an Agent and Querying a Status (2:10–2:55)

**VISUAL:** Click **Install** on the agent card, then click **Launch Agent**. The agent runner interface appears inline in the modal with quick-action buttons.

**VOICEOVER:**

> When you're ready, hit **Install**, then **Launch Agent**. CapeControl creates an authorised session for this agent and shows you the runner interface right here — no switching between tools.

**VISUAL:** Click the **"Project Status"** quick insight button. A loading indicator appears briefly, then the agent returns a structured JSON result showing project health, key metrics, and status summaries. The result is displayed in a clean, formatted view.

**VOICEOVER:**

> Let's ask it something. We'll click **Project Status**. Within seconds, the agent queries your live data, processes it, and returns a clear summary — project health, blockers, completion rates — all in one view.
>
> Behind the scenes, every interaction is cost-tracked, token-metered, and logged to your audit trail.

---

## SCENE 6 — Executing an Action and Seeing the Result (2:55–3:25)

**VISUAL:** Click the **"Top Blockers"** action button. The agent returns a structured response highlighting blockers with severity and recommended next steps. Then navigate back to the **Dashboard**. The **Activity Feed** shows new entries for the agent run.

**VOICEOVER:**

> Now let's have the agent take action. We'll run **Top Blockers** — the agent analyses your project data and returns prioritised issues with recommended next steps.
>
> Head back to your Dashboard — and there it is in your Activity Feed. Every agent run, every result, tracked in real time.

**VISUAL:** Hover over the Usage Progress Card — the API calls meter has incremented, the agent runs count has ticked up.

**VOICEOVER:**

> Your usage meters update automatically, so you always know exactly where you stand against your plan limits.

---

## SCENE 7 — Security Recap and Closing (3:25–3:45)

**VISUAL:** Split screen showing three key points with icons:
1. Lock icon — "Authorised access only"
2. Shield icon — "Full audit trail"
3. Meter icon — "Usage-metered and cost-controlled"

Then transition to a closing card with the CapeControl logo, documentation URL, and support contact.

**VOICEOVER:**

> To recap: every AI agent on CapeControl is authorised with explicit permissions, every interaction is logged in your audit trail, and every call is metered against your plan — so you're always in control of both access and cost.
>
> To get started, visit our documentation at **cape-control.com/docs**, or reach out to our support team from the **Help** page. We're here to help you get the most from your AI agents.
>
> Thanks for watching.
