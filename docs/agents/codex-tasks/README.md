# Codex Tasks (Work Briefs)

Each task file should include:

1) Goal (single sentence)
2) Constraints / safety gates (autorisen-only; no capecraft unless explicit)
3) Preflight commands (git status/branch/log)
4) Implementation steps
5) Verification commands (tests/build/lint as appropriate)
6) Expected outputs
7) Commit message

Recommended naming:
- `task-###-<slug>.md`
