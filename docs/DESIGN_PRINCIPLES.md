# CapeControl Design Principles

## 1. Calm Over Clever

CapeControl prioritizes reassurance and clarity over visual cleverness.

## 2. Reframing Over Features

We lead with anxiety reduction and perception shifts, not feature lists.

## 3. 80% Predictable / 20% Delight

Familiar structure first, with carefully placed moments of surprise.

## 4. Explainability Builds Trust

Every system action should be understandable and observable.
