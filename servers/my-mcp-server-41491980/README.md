# my-mcp-server-41491980

A new MCP server.

## Setup

1. Use the shared virtual environment:

   ```bash
   source ../../.venv/bin/activate
   ```

2. Install dependencies (if not already installed):

   ```bash
   pip install -r requirements.txt
   ```

3. Run the server:

   ```bash
   python src/server.py
   ```
